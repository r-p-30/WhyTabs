# WhyTab – Save Tabs With Purpose

WhyTab is a lightweight and purposeful browser extension to help you **save tabs with context**—intent, tags, reminders—and never lose track of *why* you opened something in the first place.

---
## Core Functionality

- **Save Tabs with Context**  
  Add notes, tags, and optional reminders when saving a tab. If a tab is already saved, you can mark it as read directly from the popup.

- **Side Panel for Managing Saved Tabs**  
  Open a dedicated side panel (shortcut: `Ctrl + Y`) to view all saved tabs. Each entry displays its title, note, tags, reminder, and the date it was saved. You can open tabs, apply filters, or mark them as read/archived.

- **Reminders & Notifications**  
  Set reminders on tabs. At the chosen time, the browser will notify you with an option to open, mark as read, or snooze the tab for the next day.

---

## Planned Enhancements

1. **Quick Save Shortcut**  
   Save the current tab instantly with default settings using a keyboard shortcut.

2. **Bulk Actions**  
   Ability to mark all filtered tabs as read in one click.

3. **Auto-Cleanup**  
   A background process to automatically remove archived tabs older than a week.

4. **Advanced Filtering**  
   Add more flexible filter options such as by date ranges, read status, or reminder presence.

5. **Cross-Browser Support**  
   Extend support beyond Chrome to browsers like Firefox, Edge, and Safari.

6. **Sync Across Devices**  
   Enable syncing of saved tabs and settings across multiple devices via cloud storage.

## Tech Stack

- JavaScript (Manifest V3)
- Chrome Extension APIs (Storage, Notifications, Side Panel)
